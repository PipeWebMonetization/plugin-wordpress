== Description ==

The Pipe Web Monetization aims to make it easy for worpress website owners to activate monetization for their sites, and link to a dashboard system that displays their earnings.
Through payment indicators, we will share the revenue according to the probability you want.
Our plugin is connected with a dashboard that will display the amounts received through web monetization. For this to happen, we collect the values ​​received through payment events, and send them to an external database, so that Pipe Dashboard can access the values ​​received at each payment pointer.

== Installation ==

1. Set your Plugin ID in dashboard (https://dashboard.pipewebmonetization.com/)
2. Add your Plugin ID in plugin settings tab
3. Add one or more payment pointers
